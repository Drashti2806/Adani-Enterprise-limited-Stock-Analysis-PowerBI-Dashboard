# Adani-Enterprise-limited-Stock-Analysis-PowerBI-Dashboard
📊 Adani Stock Data Analysis Dashboard

🚀 Project Overview

This project is an interactive Stock Market Dashboard built to analyze the performance of Adani Enterprises Limited using key financial metrics and visual insights.

The dashboard helps users understand stock trends such as opening price, closing price, volume, and 52-week highs/lows through dynamic visualizations.

📌 Features

📅 Dynamic Filters: Day, Month, Quarter, Year selection

📈 Key Metrics:

Average of 52-week High

Average of 52-week Low

Total Open & Close values

Total Volume 

📊 Visualizations:

Quarterly comparison of 52-week highs & lows

Daily stock price trends

Monthly volume analysis

Open vs Close price comparison by day

🛠️ Tools & Technologies Used

Microsoft Power BI – Dashboard creation

Microsoft Excel – Data preprocessing

Data Visualization Techniques (Bar Chart, Line Chart, Area Chart)


📊 Key Insights
Stock prices show fluctuating trends across days and months

Certain months contribute significantly to higher trading volume

Noticeable differences between open and close prices

Quarterly analysis highlights variations in 52-week highs and lows

📂 Dataset

The dataset includes:

Date-wise stock data

Open, Close prices

Volume traded

52-week High & Low values

Purpose
